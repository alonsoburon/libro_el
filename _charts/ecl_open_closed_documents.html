Open documents
Lines change, statuses shift, amounts adjust. Can be hard-deleted before posting.
Strategy:
Full replace every run
Small, bounded set
e.g. draft invoices, pending orders
Closed documents
Frozen after posting. In many jurisdictions, modifying a closed invoice is illegal.
Strategy:
Cursor-based incremental
Large, growing set, immutable
The rare case where a soft rule is backed by a hard rule (the law)
Extraction split
Run both strategies in the same pipeline. Open = full replace (small set). Closed = incremental (growing set).
