Statelessness × idempotency
Two independent properties. The goal is both.

Sweet spot
Stateless window + MERGE or dedup appends. Fresh machine, same result twice.
GOAL
Dangerous
Stateless window + raw INSERT. No state needed, but retries create duplicates.
Common
Cursor-based + MERGE or dedup appends. Needs prior state, but retries are safe.
Worst case
Cursor-based + raw INSERT. Needs state AND retries create duplicates.
Note
Full replace is stateless + idempotent by construction -- no matrix needed.
