Backfill type taxonomy
Three types, each with different scope and state-reset implications
Date-range
Reload a specific date range using partition swap or rolling window.
Scope: bounded time window
State reset: cursor for that range only
Full-table
Reload the entire table. Reset all cursor state. Start from scratch.
Scope: everything
State reset: all cursors cleared
Selective
Reload specific records by primary key. Surgical fix for known bad rows.
Scope: explicit PK list
State reset: none
