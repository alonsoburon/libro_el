Framework 01
Conforming vs transforming
Every operation during the crossing is one or the other
Conforming
Belongs in ECL -- survive the crossing
Type casting (VARCHAR → TIMESTAMP)
Null handling (preserve source nulls)
Timezone normalization (tag, don't convert)
Charset encoding (UTF-8)
Metadata injection (_extracted_at)
Synthetic key generation
Cursor-borrowing joins
Transforming
Belongs downstream -- changes business meaning
Business logic (revenue = qty × price)
Filtering (exclude inactive customers)
Enrichment joins (denormalization)
JSON flattening into relational tables
Aggregation
Decision test
If a join adds a column the business cares about → transforming
If it adds a column only the pipeline cares about → conforming
